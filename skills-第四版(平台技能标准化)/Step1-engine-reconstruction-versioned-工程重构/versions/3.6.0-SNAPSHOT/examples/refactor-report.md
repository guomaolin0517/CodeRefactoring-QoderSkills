```

- 移动模块数：{count}
- 创建容器 POM 数：{count}
- 更新 POM 文件数：{count}
- 更新 Java 文件数：{count}
- 更新配置文件数：{count}

## 编译验证
- 状态：通过/失败
- 错误数：{count}

## 编译错误（如有）
| 模块 | 文件 | 行号 | 错误类型 | 错误信息 |
|------|------|------|----------|----------|
...

## 需人工处理
| 模块 | 原因 | 建议 |
|------|------|------|
| guangdong | 无法自动识别归类 | 请手动归类 |
...
```
